Tip Calculator

This is a simple and interactive Tip Calculator built using HTML, CSS, and JavaScript. It allows users to:

    Enter the total bill amount.
    Adjust the tip percentage using a slider.
    Select the number of people sharing the bill.
    Calculate the total tip, total bill (including tip), and the amount each person needs to pay.
    Reset the calculator using the "AC" button.

Features

    Dynamic slider values display.
    Input validation for accurate calculations.
    Fully responsive design for various devices.

Technologies Used

    HTML
    CSS
    JavaScript
